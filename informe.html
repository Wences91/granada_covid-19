Introducci√≥n

El objetivo de esta notebook es visualizar los datos disponibles de la incidencia del COVID-19 en la provincia de Granada. Cabe destacar que por este motivo no se pretende realizar un an√°lisis de los mismos sino ofrecer gr√°ficos complementarios a los disponibles en el informe diario oficial a modo de fotograf√≠a de la situaci√≥n actual.

Metodolog√≠a

Los materiales empleados proceden de la Junta de Andaluc√≠a, los cuales son actualizados a diario. A excepci√≥n de los casos nuevos, el resto de variables (fallecimientos, curados, hospitalizados e ingresos en UCI) aparecen recogidos como valores acumulados, por lo que estos casos han requerido ser disgregados. No obstante, en los valores obtenidos de curados y hospitalizados diarios exite un d√≠a que dicha diferencia de acumulados da como resultado un valor negativo.

datos <- read.csv2('datos.csv', stringsAsFactors = FALSE)

Partiendo de las medidas antes citadas se han realizados gr√°ficos con la evoluci√≥n diaria y acumulada, incluyendo en ambos la media m√≥vil de 3 d√≠as. Todos ellos se han realizado tomando como punto de inicio el d√≠a 13 de marzo, cuando se confirmaron los primeros positivos en la provincia. Tanto para la disgregaci√≥n como para la visualizaci√≥n, elaborada mediante ggplot2, se han creado las funciones est√°n incluidas a continuaci√≥n. Al respecto cabe destacar que tanto los datos usados como este archivo Rmd se encuentran disponibles en GitHub.

# funci√≥n de disgregaci√≥n
diario <- function(datos, columna){
  aux <- datos[which(datos$Medida == columna),]
  aux <- aux[order(aux$Fecha, decreasing = FALSE),]
  aux$Valor[-1] <- aux$Valor[-1] - aux$Valor[-dim(aux)[1]]
  
  return(aux)
}

# funci√≥n para calcular la media movil de n d√≠as
media_movil <- function(datos, dias){
  datos$Valor <- filter(datos$Valor, rep(1/dias, dias), sides=2)
  
  return(datos)
}

# funci√≥n para dibujar la evoluci√≥n diaria
evolucion <- function(datos, titulo, color){
  ggplot(datos, aes(x = Fecha, y = Valor)) +
  geom_col(fill = color) +
  scale_x_date(date_breaks = '2 days', date_labels = '%d-%b', limits = c(datos$Fecha[17], NA)) +
  labs(title = titulo, x = '', y = '') +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 90, vjust=0.5, hjust=1),
        plot.title = element_text(hjust = 0.5))
}

# funci√≥n para dibujar la evoluci√≥n diaria y media movil
evolucion_movil <- function(datos1, datos2, titulo, serie, color){
  ggplot() +
    geom_col(data=datos1, aes(x = Fecha, y = Valor, fill = serie)) +
    geom_line(data=datos2, aes(x = Fecha, y = Valor, color = 'Media movil'), alpha = 0.7, size = 1.2) +
    scale_fill_manual('', values = color) +
    scale_color_manual('', values = 'black') +
    scale_x_date(date_breaks = '2 days', date_labels = '%d-%b', limits = c(as.Date('12-03-2020', format = '%d-%m-%Y'), NA), expand = c(0,0)) +
    labs(title = titulo, x = '', y = '') +
    theme_minimal() +
    theme(axis.text.x = element_text(angle = 90, vjust=0.5, hjust=1),
          plot.title = element_text(hjust = 0.5),
          legend.position = 'bottom')
}
